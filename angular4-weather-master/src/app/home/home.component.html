Тестовое задание

Реализовать компонент который будет получать данные о погоде из API http://openweathermap.org/api, и отображать их в произвольном виде (на Ваше усмотрение).

Обязательные данные для вывода:

Название города; Иконка (загрузка иконки из http://openweathermap.org); Описание погоды (‘description’); Краткое описание погоды (‘main’); Облачность (‘clouds’);

Требования:

Получать данные о погоде на текущий момент, город Киев; Данные должны обновляться каждые 20 секунд и также по нажатию кнопки; Организовать проект с помощью Angular CLI; Использовать Angular Material; Структурировать код для возможность расширения и удобства переиспользования.
